### 序列博弈
>游戏介绍
- 每次游戏会随机生成一个 <kbd>范围</kbd> 与 <kbd>倍率</kbd>
- 最终答案为所有灵感者所选数字 **总和** **平均** 数再乘以 **倍率**  
- 获胜者为与答案最接近的灵感者
- 如果获胜者有多个积分奖励将会平分

$$\frac{\sum_{i=1}^{n} x_{i} }{n}\ · 倍率$$  

![](https://pic1.imgdb.cn/item/644491290d2dde5777178920.png)  

**参数列表**

|  参数   | 含义  |
|  ----  | ----  |
|  n  | 参加游戏人数 |
|  x  | 每个游戏参与者所选数字 |
|  范围  | 在给定范围内选择数字 |
|  倍率  | 要接近的平均数的倍率 |
